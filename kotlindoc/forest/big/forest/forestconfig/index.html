ForestConfig

data class ForestConfig: Any

An object that stores the configuration of a Forest.

Fields

Name Description
var level: Level

The logging level. This sets what level of log to be passed to a Tree.
var preProcessLog: PreProcessLogCallback?

A callback to be invoked before passing a LogEntry to the Tree. This is useful to filter or modify a LogEntry before the log is being passed to Tree.log.
var allowGlobalOverride: Boolean

Setting true will allow any configuration change from Forest.Global to override this Forest, false will prevent this override.

Methods

plant

fun plant(tree: Tree)

Plant the tree to this Forest. Any logging method call (i.e Forest.d, Forest.i, etc) will be forwarded to this tree if the Forest determine it should log the given message.

Parameters

Name Description
tree: Tree

The Tree that will handle the logging call.

ReturnValue

Name Description
Unit

cut

fun cut(tree: Tree)

Cut the tree from this Forest. Any logging method call (i.e Forest.d, Forest.i, etc) will no longer be forwarded to this tree.
