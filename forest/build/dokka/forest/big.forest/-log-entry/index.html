LogEntry

data class LogEntry

An object that holds the information of a log.

Parameters

level - The logging level of the log. the Forest received a logging call.

context - The ForestContext that contains the data from the Forest.

message - The message to be logged.

tag - The tag. The value is either the name of a Forest set from getForest or the class name of the first stacktrace from the throwable, if any.

throwable - The Throwable to be logged.

attributes - A collection of attributes to be added to this log.
