preProcessLog

open fun preProcessLog(callback: PreProcessLogCallback): Unit

Set the PreProcessLogCallback to this Forest. This will allow the Forest to send the LogEntry every time before it's sent to the Trees. You can use this callback to filter or modify a LogEntry before the LogEntry is being sent to the Trees in this Forest.

Parameters

callback - The PreProcessLogCallback to be set into this Forest.
