ForestConfig

data class ForestConfig

An object that stores the configuration of a Forest.

Parameters

level - The logging level. This sets what level of log to be passed to a Tree.

preProcessLog - A callback to be invoked before passing a LogEntry to the Tree. This is useful to filter or modify a LogEntry before the log is being passed to Tree.log.

allowGlobalOverride - Setting true will allow any configuration change from Forest.Global to override this Forest, false will prevent this override.

trees - A collection of Trees to be added to the Forest.
