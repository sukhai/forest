ForestDataContext

class ForestDataContext: ForestContext, Any

A type of ForestContext that store data in a ConcurrentHashMap.

You can instantiate an instance of this class through ForestContext.createDataContext.

Name Description
val size: Int

Return the number of data in the context.
val keys: Set<String>

Return all the keys from the context.

fun put(key: String, value: Any): Any?

Maps the given key to the specified value in this context. Neither the key nor the value can be null. The value can be retrieved by calling the get method with a key that is equal to the original key.

Parameters

Name Description
key: String
