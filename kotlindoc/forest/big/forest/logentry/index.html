LogEntry

data class LogEntry: Any

An object that holds the information of a log.

Fields

Name Description
val level: Level

The logging level of the log. the Forest received a logging call.
val message: String

The message to be logged.
val context: ForestContext

The ForestContext that contains the data from the Forest.
val tag: String?

The tag. The value is either the name of a Forest set from getForest or the class name of the first stacktrace from the throwable, if any.
val throwable: Throwable?

The Throwable to be logged.
val attributes: Map<String, Any>

A collection of attributes to be added to this log.
