Forest

interface Forest

An object that allows the user to send any log entry to a group of Trees.

You can create or get an implementation of this interface through Forest.getForest.

Types

Global

A singleton class that represents the global Forest. This global Forest holds a collection of Forests that are created through getForest.

companion object Global : AbstractForest

Level

A class that represents the logging level. A Forest will use the level to determine if it should log a given LogEntry.
