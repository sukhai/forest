set

open operator fun set(key: String, value: Any): Unit

Maps the given key to the specified value in this context. Neither the key nor the value can be null. The value can be retrieved by calling the get method with a key that is equal to the original key.

Parameters

key - The key with which the specified value is to be associated.

value - The value which is associated with the key.

Return
The previous value associated with key, or null if there was no mapping for key.
